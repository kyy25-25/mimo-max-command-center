# Sample Mission

## User goal

Create a stronger Xiaomi MiMo Orbit project after an earlier submission received Pro, with the new target being Max.

## Agent outputs

### Signal Scout

- Problem: previous project looked useful but not differentiated enough for Max.
- Opportunity: position the new submission as an agentic workflow operating system.
- Requirement: include UI, CLI proof, docs, audit layer, and clear MiMo integration path.

### Strategy Planner

Milestones:

1. Build dashboard UI.
2. Build CLI proof run.
3. Generate JSON evidence report.
4. Write Max application copy.
5. Capture screenshots and push to GitHub.

Acceptance tests:

- `npm run build` passes.
- `npm run proof` writes `data/latest-proof-report.json`.
- README explains why the project is Max-level.
- Application text avoids placeholder claims.

### Evidence Auditor

- Hallucination risk: low, because repo artifacts are inspectable.
- Privacy risk: low, because demo mode uses no private credentials.
- Evidence quality: high, because proof report and screenshots can be uploaded.

### Submission Packager

Recommended form narrative: emphasize five-agent orchestration, auditability, high-volume usage, and MiMo API integration path.
